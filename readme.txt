This plugin adds custom conditions to Breakdance Builder allowing users to control elements' visibility based on cookie values.

== Description ==

Breakdance Custom Conditions and Blocks for Cookie Storage and More is a WordPress plugin developed by Edoardo Guzzi and An Idea For Business. It extends the functionality of Breakdance Builder by introducing a custom condition that enables users to show or hide elements on their website based on the presence and value of cookies.

== Installation ==

1. Upload the entire `breakdance-extend-block-and-conditions-for-cookie-storage-and-more` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= How to use this plugin? =

After installing and activating the plugin, a new condition titled "Controllo Cookie" will be available in Breakdance Builder. Users can utilize this condition to control the visibility of website elements based on cookie values.
